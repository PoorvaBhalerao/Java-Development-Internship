## Library Management System

## Objective
To develop a Java-based console application that manages library operations such as adding, searching, issuing, returning, viewing, and removing books using Object-Oriented Programming principles and Java Collections Framework.

## Features Implemented
1.Add a new book
2.View all books
3.Search book by ID
4.Search book by title
5.Issue a book
6.Return a book
7.Remove a book
8.Book availability tracking
9.Duplicate Book ID validation
10.0Menu-driven user interface

## Technologies Used
* Java
* OOP (Object-Oriented Programming)
* HashMap Collection
* Scanner Class
* Java Collections Framework

## File Structure
