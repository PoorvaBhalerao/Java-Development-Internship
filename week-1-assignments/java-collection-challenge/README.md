# Java Collections Challenge

## Objective

This project demonstrates the practical usage of Java Collections Framework by implementing operations using ArrayList, HashMap, and Queue. It helps in understanding data storage, retrieval, updating, searching, and iteration using different collection types.

## Features Implemented

### ArrayList (Book Management)

* Add Book
* Remove Book
* Update Book
* Search Book
* Display All Books

### HashMap (Student Marks Management)

* Add Student Record
* Retrieve Marks by Student ID
* Update Marks
* Check Student Existence
* Display All Student Records

### Queue (Customer Queue Management)

* Add Customer
* Serve Customer (FIFO)
* Peek Next Customer
* Display Queue Contents

## Technologies Used

* Java
* Java Collections Framework

  * ArrayList
  * HashMap
  * Queue
  * LinkedList
* Exception Handling
* OOP Concepts
