# Banking Application

## Objective

Develop a console-based Banking Application using Core Java that allows users to perform basic banking operations such as account creation, deposit, withdrawal, balance inquiry, transaction history tracking, and account closure.

## Features Implemented
* Create Bank Account
* Deposit Money
* Withdraw Money
* Balance Inquiry
* Transaction History
* PIN Verification
* Minimum Balance Validation
* Account Closure
* Exception Handling for Invalid Inputs
* Menu-Driven Interface

## Technologies Used
* Java
* Object-Oriented Programming (OOP)
* ArrayList Collection Framework
* Exception Handling
* Scanner Class

## File Structure
BankingApplication.java
│
├── BankAccount.java
├── Banking.java
└── BankingApplication.java

## Steps to Compile and Run
1. Open terminal/command prompt.
2. Navigate to the project directory.
3. Compile the program:
javac BankingApplication.java
4. Run the program:
java BankingApplication
